# How To Play

# Dedicated Server
<img width="1287" height="755" alt="launcher" src="https://github.com/user-attachments/assets/53eab04d-9666-43b5-84a1-354af6325d3c" />

*For Dedicated Play (Also Supports RADMIN Vpn)*
1. Enter your name.
2. Server IP.
3. Port (Default to 7777).
4. Click Confirm

# Steam
<img width="1286" height="752" alt="image" src="https://github.com/user-attachments/assets/e47823c1-0b48-4c96-9cf9-135474ce2a2f" />

*For Steam Relay*
1. Add Non-Steam game to steam, add this game's exe.
2. Have Online Status
3. Click Steam

After that, Click Create Lobby, you can invite your friends to your lobby using steam chat, or copy room code and paste it to join. Game starts when everyone is ready.

<img width="1285" height="753" alt="image" src="https://github.com/user-attachments/assets/a9b45895-5f40-4164-a4c3-7aaa76ad6a5f" />

Join One of the two teams, match starts when both teams have atleast one player. Its a Deathmatch



# Mechanics
<img width="1285" height="748" alt="image" src="https://github.com/user-attachments/assets/b4f895ec-47df-41ca-9c82-c8e95bcf5c50" />

Use W - A - S - D to move, Space to Jump, Left Alt to swap shoulders, Right click to enter Aiming mode.
For Shooting, You have to hold Mouse1 (Left Mouse Btn) to Charge a shot, release to shoot it.
